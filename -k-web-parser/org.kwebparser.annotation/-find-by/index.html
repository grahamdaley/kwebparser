FindBy

@Target(allowedTargets = [AnnotationTarget.PROPERTY, AnnotationTarget.CLASS, AnnotationTarget.FILE])
annotation class FindBy(val id: String = "", val name: String = "", val className: String = "", val css: String = "", val tagName: String = "", val linkText: String = "", val partialLinkText: String = "")

Used to mark a property on a Page Object to indicate an alternative mechanism for locating the element or a list of elements.

You need to specify one of the location strategies (eg: "id") with an appropriate value to use.

For example:

@FindBy(id = "foobar") Element foobar;
*

and this points to a list of elements:

@FindBy(tagName = "a") Elements links;
*
